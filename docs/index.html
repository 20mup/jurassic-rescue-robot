Jurassic Rescue Robot

Autonomous + Manual Robot for Rescue Missions

Overview

This robot was designed and built to complete a Jurassic-themed rescue mission using both autonomous and manual modes. With constraints on the number of motors and fabrication tools, the team created a custom mechanism capable of gate control and object retrieval while navigating a complex course.

Final Design Features

  • Manual Control: Joystick-based drive with feedback
  • Autonomous Mode: Ultrasonic-guided navigation
  • Claw Mechanism: For figure rescue and retrieval
  • Chassis: Hybrid 3D-printed + laser cut base
  • Specialized Caster: Enables full maneuverability

Tech Stack

  • Python – Raspberry Pi Pico control scripts
  • C++ – Arduino-based logic
  • SolidWorks – CAD modeling

Lessons Learned

  • Prototyping with cardboard drastically reduced rework
  • Closed-loop feedback increased driving precision
  • Scrum-based planning accelerated deliverable quality
